# ConcoctJS-Markdown

Markdown plugin for [ConcoctJS](https://github.com/EyalAr/ConcoctJS/).

## Parameters:

### `fields`

**Type:** String / Array  
**Required:** yes

Which fields in the context to compile.

### `highlight``

**Type:** Boolean  
**Required:** no

If `true`, uses [`highlight.js`](https://github.com/isagalaev/highlight.js) to highlight code blocks.
